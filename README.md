# Mohammad Yakub - Developer Portfolio

A modern, responsive, single-page developer portfolio featuring a dynamic matrix "digital rain" background, custom cursor, glassmorphism UI elements, and glowing neon accents.

### 🌐 Live Demo
**[View the Live Portfolio Here](https://yakub-portfolio.vercel.app/)**

---

## 🚀 Features
- **Immersive Design:** Futuristic Hacker/Matrix aesthetic with a deep dark gradient and neon green elements.
- **Dynamic Live Background:** Continuous "0" and "1" falling digits powered by an optimized HTML5 Canvas & JavaScript loop.
- **Interactive UI:** A custom-built animated cursor that responds to interactive elements.
- **Responsive Layout:** fully adaptable `Grid` and `Flexbox` architecture ensures pixel-perfect rendering across mobile devices and large monitors.
- **Working Contact Form:** Direct email routing integration via FormSubmit API.

## 🛠️ Tech Stack
This project was designed and built purely from scratch using core web technologies without heavy frameworks:
- **HTML5** (Semantics & Structure)
- **Vanilla CSS3** (Custom Properties, Grid, Flexbox, Animations)
- **Vanilla JavaScript** (DOM Manipulation, Canvas Rendering, Observers)

## 📁 Sections
1. **Hero:** Introduction and animated typing effect.
2. **About Me:** Core traits, skills, and track record.
3. **Tech Stack:** Grid layout of known programming languages and tools with visual proficiency bars.
4. **Projects:** Showcased Python and Automation logic projects.
5. **Academic Journey:** Education timeline.
6. **Contact:** Fully functional messaging form with direct-to-inbox forwarding.

## 💻 Local Setup
To run this project locally, simply clone the repository and open `index.html` in any modern web browser or use a local development server like VS Code Live Server.

```bash
git clone https://github.com/Yakub-bmsce/yakub-portfolio.git
cd yakub-portfolio
```
